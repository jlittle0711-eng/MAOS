\# MAOS — MakerHub Autonomous Operating System



MAOS is a signal-aware decision engine for analyzing grant opportunities and prioritizing actions.



\## Core Features



\- Signal Processing Layer (external intelligence integration)

\- Decision Engine (priority + action classification)

\- Centralized Controller (single execution entry point)

\- Explainable AI outputs (reason + flags)



\## System Architecture



Controller → Signal Processor → Decision Engine → Output



\## How to Run



```bash

cd core

python maos\_controller.py
